キーボードショートカット

磁石モード

オブジェクトを移動させた際に同じ種類のオブジェクトを巻き込みながら移動させる

オブジェクトをマウス左ボタン長押し+ドラッグ操作
長押しで磁石モードを開始し、ドラッグでオブジェクトを移動
任意の場所をクリック
磁石モードを解除

範囲選択モード

選択領域の内側に見えているオブジェクトを選択状態にする。

テーブルをマウス左ボタン長押し+ドラッグ操作
長押しで範囲選択モードを開始し、ドラッグで範囲領域を指定
[shift]+マウス左ボタン+ドラッグ操作
即時に範囲選択モードを開始し、ドラッグで範囲領域を指定
[ctrl]+マウス左ボタン
クリックしたオブジェクトの選択状態を切り替える
[ctrl]+マウス左ボタン+ドラッグ操作
マウスカーソルの触れた対象を選択状態にする
選択状態のカードを移動
選択状態のすべてのオブジェクトを動機して移動
選択状態のカードを回転
選択状態のすべてのオブジェクトを動機して回転
選択状態のカードを右クリックして「選択したカード」のコンテキストメニューを開く
選択状態のすべてのカードを「表にする」「裏にする」「自分だけ見る」が可能
選択状態のデッキを右クリックして「選択した山札」のコンテキストメニューを開く
選択状態のすべてのカードを「表にする」「裏にする」「自分だけ見る」「シャッフル」が可能
選択状態のキャラクターを右クリックして「選択したカード」のコンテキストメニューを開く
選択状態のすべてのキャラクターを「共有インベントリ」「個人インベントリ」「墓場」に移動が可能
任意の場所をクリック
全ての選択状態を解除
