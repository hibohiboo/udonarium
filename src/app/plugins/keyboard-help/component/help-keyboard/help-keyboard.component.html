キーボード

カード

カードの上で[t]
カードを横にする
カードの上で[u]
カードを縦にする

地形

※対象の地形をクリックしてからキーボードを押してください。

[{{constants.terrainCopyKey}}]
地形をコピー
[{{constants.terrainDeleteKey}}]
地形を削除
[{{constants.terrainEditKey}}]
地形設定を編集

テーブル

[→]
テーブルを左に移動
[↑]
テーブルを下に移動
[←]
テーブルを右に移動
[↓]
テーブルを上に移動
[shift]+[→]
テーブルを時計回りに回転
[shift]+[↑]
テーブルを手前に倒す
[shift]+[←]
テーブルを逆時計回りに移動
[shift]+[↓]
テーブルを奥に倒す
[?]
キーボードショートカット表示
[ESC]
キーボードショートカットを閉じる
